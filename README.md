# Bank Term Deposit Prediction

## 📌 Project Overview
Predict whether a customer will subscribe to a term deposit.

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost

## 📊 Key Features
- Feature Engineering (Age Group, Balance Group)
- Model Training & Evaluation
- Business Insights

## 📈 Results
- Best Model: XGBoost
- Accuracy: 82%

## 💼 Business Impact
Helps banks target potential customers and improve marketing ROI.
